## Insert SVG

Insert SVGs directly into the dom with js.

#### Usage

Simply place the path to the svg within the data attribute.

```html
<div class="my-class" data-svg="img/cta.svg"></div>
```
Call the function on the class you choose.
```js
insertSvg('my-class');
```
